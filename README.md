Setup
=======================================
## group pcs
* user: group1 ... group4
* pw: group1 ... group4
* chechout ipa_seminar to ~/git
* ROS_PACKAGE_PATH=~/git/ipa_seminar:/opt/ros/groovy/share:/opt/ros/groovy/stacks

## working environment
* editor: gedit
* terminal: terminator
* aliases: 
  * group (default) --> ROS_PACKAGE_PATH for group
  * solution --> ROS_PACKAGE_PATH for solution
